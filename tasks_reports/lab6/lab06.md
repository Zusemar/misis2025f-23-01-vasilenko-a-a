# Лабораторная работа №6: Векторизация границ объектов

## Краткое описание

Лабораторная работа по векторизации границ объектов интереса на изображениях.

## Основные функции

1. **Нахождение и векторизация границ объектов интереса**
   - Поиск контуров объектов
   - Упрощение контуров алгоритмом Douglas-Peucker
   - Сохранение векторного представления

2. **Генерация эталона векторного представления**
   - Автоматическое создание эталона
   - Сохранение в текстовом формате
   - Возможность ручной корректировки

3. **Визуализация векторного представления**
   - Визуализация контуров
   - Визуализация ошибок (TP, FP, FN)

4. **Измерение качества векторизации**
   - Метрики Precision, Recall, F1
   - Расстояние Хаусдорфа
   - Среднее расстояние между контурами
   - IoU для контуров

## Использование

### Часть 1: Векторизация
```bash
./build/prj.lab/lab06/task06 part1 walnut01.0130_converted.png output/
```

### Часть 2: Генерация эталона
```bash
./build/prj.lab/lab06/task06 part2 walnut01.0130_converted.png output/
```

### Часть 3: Оценка качества
```bash
./build/prj.lab/lab06/task06 part3 walnut01.0130_converted.png output/ground_truth_contours.txt output/
```
