# Лабораторная работа №5: Детектирование объектов

## Задание

Реализовать функционал детектирования объектов:
1. Локализация объектов интереса (анализ компонент связности, выбор подходящего масштаба)
2. Оценка достоверности локализации
3. Анализ на нескольких масштабах (если необходимо)
4. Визуализация результатов и эталона детекций (в т.ч. отклонений от эталона и ошибок)
5. Измерение качества детектирования
6. Изготовление эталона детектирования

## Исходные данные

- Изображение: прикрепленный файл (темное изображение с низким контрастом)
- Расположение: `prj.lab/lab05/`

## Реализованные методы

### 1. Локализация объектов интереса
- Адаптивная бинаризация для выделения объектов
- Анализ компонент связности
- Многомасштабный анализ (пирамиды изображений)
- Морфологические операции для очистки
- Фильтрация по размеру объектов

### 2. Оценка достоверности локализации
- Confidence score на основе:
  - Контраста объекта относительно фона
  - Отношения площади объекта к размеру bounding box
  - Компактности объекта (отношение площади к периметру)

### 3. Метрики качества детектирования
- Precision, Recall, F1-score
- Mean IoU (Intersection over Union) для правильно детектированных объектов
- TP, FP, FN подсчеты

### 4. Визуализация
- Bounding boxes с цветовой кодировкой по достоверности
- Сравнение с эталоном (правильные детекции, ложные срабатывания, пропущенные объекты)

## Результаты

Все результаты сохраняются в указанной директории:
- Детектированные объекты с визуализацией
- Эталон детектирования
- Метрики качества (CSV)
- Визуализация ошибок детектирования
- Анализ компонент связности
